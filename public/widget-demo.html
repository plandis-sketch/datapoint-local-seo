WordPress + Elementor
WordPress (Gutenberg)
Any HTML Site
1
Log into WordPress
Go to yoursite.com/wp-admin and log in with your admin credentials.
2
Navigate to the target page
In the left sidebar, go to Pages and find the page where you want the review widget to appear (e.g. the homepage or a landing page).
3
Open Elementor
Click the blue "Edit with Elementor" button at the top of the page. Wait for the Elementor editor to fully load.
4
Add an HTML widget
In the left panel, search for "HTML" in the widget search bar. You'll see a widget called "HTML" (with a </> icon). Drag it onto your page in the exact spot where you want the reviews to show up.
5
Paste the embed code
Click on the HTML widget you just placed. In the left panel, you'll see a "HTML Code" text area. Paste the embed code you copied from Step 3 above into this text area.
6
Publish
Click the green "Update" button at the bottom of the Elementor panel to save and publish your changes.
The review widget will NOT render inside the Elementor editor preview — that's completely normal. It only loads on the live, published page because it needs to fetch data from our review server.
7
Verify on the live site
Visit the live page (not the editor). You should see the review carousel with the client's Google reviews, star ratings, and navigation arrows. If it doesn't appear, open your browser's Developer Tools (F12) and check the Console tab for any error messages.

4
Paste the embed code
Paste the embed code from Step 3 above directly into the Custom HTML block's text area.
5
Update the page
Click "Update" in the top right to publish. Then visit the live page to verify the widget loads.
Just like Elementor, the preview inside the editor won't show the widget. Check the live page.

2
Paste the embed code
